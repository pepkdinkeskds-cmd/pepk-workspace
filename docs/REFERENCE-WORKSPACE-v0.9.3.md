# PEPK Workspace v0.9.3 — Referensi Workspace

## Perubahan

Halaman Ruang Kerja memiliki lima tab:

1. Perencanaan
2. Evaluasi
3. Pelaporan
4. Keuangan
5. Referensi

Referensi membaca resource dari Document Center dan memakai URL root `document_center_url`.

## Batasan operasional

- Referensi tidak masuk Google Form unggah.
- Referensi tidak menambah Upload_Routes.
- Referensi tidak ditampilkan sebagai kartu kelima di Beranda.
- Fungsinya adalah akses cepat ke regulasi, pedoman, profil, SOP, template, dokumentasi, dan bahan pendukung.
