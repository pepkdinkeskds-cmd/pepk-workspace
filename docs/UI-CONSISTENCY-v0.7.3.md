# PEPK Workspace v0.7.3 — UI Consistency

## Launchpad aplikasi
- Beranda dan Ruang Kerja menggunakan kartu aplikasi horizontal.
- Ruang Kerja memakai dua kolom pada layar lebar dan satu kolom pada tablet/ponsel.
- Label kategori berulang disembunyikan pada kartu compact.
- Nama panjang memakai ellipsis, bukan pemenggalan kata vertikal.

## Pustaka Materi Monev
- Form pencarian menggunakan komponen `filter-panel` yang sama dengan Pustaka.
- Label, input, dropdown, tombol Reset, border, radius, dan bayangan diseragamkan.
- Tata letak berubah menjadi tiga, dua, lalu satu kolom sesuai ruang layar.
