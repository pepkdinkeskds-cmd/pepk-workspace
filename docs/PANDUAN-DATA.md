# Panduan Pengisian Data

- `is_active`: isi `TRUE` untuk ditampilkan, `FALSE` untuk disembunyikan.
- `sort_order`: angka urutan, mulai dari 1.
- `open_mode`: isi `new_tab` untuk tautan luar.
- `type`: gunakan `document`, `reference`, `application`, atau `shortcut`.
- `workspace_id`: harus sama dengan `id` pada sheet `Workspaces`.
- `keywords` dan `aliases`: pisahkan beberapa kata dengan koma.
- Jangan memasukkan data rahasia atau pribadi.
