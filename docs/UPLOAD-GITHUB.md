# Upload PEPK Workspace v0.3.0 ke GitHub

1. Ekstrak `pepk-workspace-v0.3.0.zip`.
2. Buka repository `pepk-workspace`.
3. Pilih **Add file → Upload files**.
4. Upload seluruh isi folder `pepk-workspace-v0.3.0` ke root repository.
5. Timpa seluruh file versi lama.
6. Commit dengan pesan: `feat(release): perbarui PEPK Workspace ke v0.3.0`.
7. Tunggu GitHub Pages selesai memperbarui situs.
8. Buka situs menggunakan `Ctrl + F5`.

Import `PEPK_Workspace_Data_v0.3.0.xlsx` ke Google Sheets menggunakan opsi **Replace spreadsheet** agar tombol folder utama dan informasi terbaru tersedia dari sumber data.
