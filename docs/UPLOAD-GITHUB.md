# Upload PEPK Workspace v0.2.1 ke GitHub

1. Ekstrak `pepk-workspace-v0.2.1.zip`.
2. Buka repository `pepk-workspace`.
3. Pilih **Add file → Upload files**.
4. Upload seluruh isi folder `pepk-workspace-v0.2.1` ke root repository.
5. Pastikan folder `assets/apps` ikut terunggah.
6. Commit dengan pesan: `feat(release): perbarui PEPK Workspace ke v0.2.1`.
7. Tunggu GitHub Pages selesai memperbarui situs.
8. Buka situs menggunakan `Ctrl + F5` untuk membersihkan cache versi sebelumnya.

Sebelum atau sesudah upload, import `PEPK_Workspace_Data_v0.2.1.xlsx` ke Google Sheets agar data aplikasi dapat dikelola melalui spreadsheet.
