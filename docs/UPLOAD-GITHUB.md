# Cara Memperbarui GitHub

1. Ekstrak paket versi terbaru.
2. Buka repository `pepk-workspace`.
3. Pilih **Add files → Upload files**.
4. Pilih seluruh file dan folder dari hasil ekstraksi.
5. Aktifkan pilihan untuk menimpa file lama.
6. Isi commit message: `feat(release): perbarui PEPK Workspace ke v0.2.0`
7. Klik **Commit changes**.
8. Tunggu sekitar 1–3 menit, lalu buka GitHub Pages dan lakukan hard refresh (`Ctrl + F5`).
