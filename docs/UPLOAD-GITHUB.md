# Upload PEPK Workspace v0.4.0 ke GitHub

1. Ekstrak `pepk-workspace-v0.4.0.zip`.
2. Buka repository `pepk-workspace`.
3. Pilih **Add file → Upload files**.
4. Upload seluruh isi folder `pepk-workspace-v0.4.0` ke root repository.
5. Timpa file lama.
6. Commit dengan pesan: `feat(release): perbarui PEPK Workspace ke v0.4.0`.
7. Tunggu GitHub Pages selesai memperbarui website.
8. Tekan **Ctrl + F5**.

## Google Sheets

Import `PEPK_Workspace_Data_v0.4.0.xlsx` menggunakan opsi **Replace spreadsheet**. Pastikan akses spreadsheet tetap **Anyone with the link → Viewer**.
