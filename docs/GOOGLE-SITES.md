# Memasang PEPK Workspace di Google Sites

## URL yang digunakan

```text
https://pepkdinkeskds-cmd.github.io/pepk-workspace/?embed=1
```

Parameter `embed=1` menampilkan PEPK Workspace dalam mode ringkas supaya header Google Sites tidak bertumpuk dengan header aplikasi.

## Langkah pemasangan

1. Buka Google Sites dan masuk ke halaman tujuan.
2. Pilih **Sisipkan → Sematkan → URL**.
3. Tempel URL di atas.
4. Pilih tampilan halaman penuh atau atur tinggi area sematan secukupnya.
5. Publikasikan Google Sites dan uji melalui komputer serta HP.

## Perilaku mode ringkas

- Header dan footer utama PEPK Workspace disembunyikan.
- Toolbar kecil tetap menampilkan nama aplikasi.
- Tombol **Buka penuh** membuka PEPK Workspace pada tab baru.
- Folder Drive dan aplikasi eksternal tetap terbuka pada tab baru.
