# Template Pesan WhatsApp

Ganti teks `[URL FORM]` dengan URL yang tersimpan pada sheet `Settings`.

## Unggah dokumen

```text
📁 Unggah Dokumen PEPK

Silakan kirim dokumen melalui formulir berikut:
[URL FORM UNGGAH]

Pilih ruang kerja, kelompok dokumen, tahun, dan subfolder tujuan. Dokumen akan diperiksa administrator sebelum ditempatkan pada folder utama.
```

## Tambah agenda

```text
📅 Tambah Agenda PEPK

Silakan kirim informasi rapat atau kegiatan melalui formulir berikut:
[URL FORM AGENDA]

Agenda akan tampil di PEPK Workspace setelah diperiksa administrator.
```
