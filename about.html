Tentang

PEPK Workspace

Workspace internal untuk mempercepat akses dokumen, aplikasi, agenda, dan informasi kerja PEPK.

Tujuan Produk

Satu pintu menuju dokumen kerja PEPK

PEPK Workspace membantu staf menemukan folder kerja, membuka aplikasi penunjang, melihat agenda, dan membaca ringkasan capaian dengan cepat. Pencarian dapat menggunakan nama dokumen, aplikasi, tahun, ruang kerja, maupun kebutuhan pekerjaan.

Search-first

Pencarian menjadi jalur utama, sementara ruang kerja membantu penelusuran berdasarkan konteks pekerjaan.

Terhubung ke Drive

Resource dokumen mengarah ke folder tahun di Google Drive, sedangkan resource aplikasi membuka portal kerja terkait di tab baru.

Izin tetap terjaga

PEPK Workspace tidak menggantikan pengaturan akses. Hak membuka dokumen tetap mengikuti izin Google Drive.

Mudah diperbarui

Metadata resource dapat diperbarui melalui Google Sheets tanpa mengubah source code aplikasi.
